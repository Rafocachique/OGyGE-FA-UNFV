# **App Name**: Gestión de Tesis – OGGE – UNFV

## Core Features:

- User Authentication and Role Management: Secure authentication system with role-based access control (Admin, Revisor, Asesor, Supervisor, Alumno, Decano) to manage permissions and access to different functionalities.
- Thesis Plan Management: Allow students to submit thesis plans, and administrators to manage the thesis plan lifecycle (review, feedback, approval).
- Document Management: Enable uploading, storing, and managing documents related to thesis plans and revisions, ensuring version control and easy access.
- Automated Status Updates and Notifications: Implement a system that automatically updates the status of thesis plans and sends notifications to relevant users (students, reviewers, advisors) based on predefined events and deadlines.
- AI-Powered Progress Prediction: Implement an AI tool that analyzes thesis data (e.g. historical completion times, number of revisions required, current student progress) to predict the likelihood of on-time completion and flag potential delays.
- Reporting and Analytics: Generate reports and analytics dashboards to provide insights into thesis progress, bottlenecks, and overall program performance.
- User Dashboard: Personalized dashboards for each user role, providing a clear overview of relevant tasks, notifications, and progress metrics.

## Style Guidelines:

- Primary color: Medium orange (#F59E0B) to reflect the university's brand identity.
- Background color: Light beige (#FDF4E7), providing a soft and clean backdrop.
- Accent color: Darker orange (#F97316) for interactive elements and calls to action.
- Body and headline font: 'Roboto' (sans-serif) for UI text providing a clean, readable interface.
- Table cell font: 'Roboto Condensed' (sans-serif) for consistent tabular data presentation.
- Use clear and intuitive icons for file types (PDF, Word), calendar events, remaining time, and overdue items.
- Implement soft zebra striping in tables for improved readability.
